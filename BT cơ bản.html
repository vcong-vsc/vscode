Bài 2: Thiết kế trang Web với nội dung sau

HTML là một ngôn ngữ đánh dấu được sử dụng để tạo cấu trúc cho trang web.
Nó cho phép người lập trình dễ dàng làm nổi bật nội dung quan trọng.

Bạn có thể sử dụng thẻ chữ lớn để làm tiêu đề nổi bật và chữ nhỏ để ghi chú.

Ví dụ: Công thức toán học nổi tiếng E = mc² thể hiện mối quan hệ giữa năng lượng và khối lượng.

Trong hóa học, ta có công thức nước là H₂O, với số 2 ở chỉ số dưới.

Thẻ samp thường được dùng để hiển thị kết quả từ chương trình.

Ví dụ sử dụng thẻ pre: for (int i=0; i < 5; i++) { } printf("Hello World\n");

Nếu muốn gạch bỏ nội dung, ta dùng thẻ strike, hoặc thẻ s, hay thẻ del.

Kết luận: Kết hợp các thẻ trên giúp văn bản rõ ràng, trực quan và dễ đọc hơn.

Bài 3: Thiết kế trang Web với nội dung sau


Môn học: Cấu trúc dữ liệu và Giải thuật.
Mục tiêu của môn học là giúp sinh viên hiểu rõ về cách lưu trữ và xử lý dữ liệu.
Kiến thức này rất quan trọng và cần thiết cho lập trình viên.
Nội dung bài giảng có thể được viết bằng chữ lớn để làm tiêu đề, hoặc chữ nhỏ để ghi chú.
Ví dụ công thức toán: n² + 2n + 1 = (n+1)².
Công thức hóa học thường dùng: H₂SO₄
Kết quả khi chạy code có thể hiển thị trong thẻ samp.
Ví dụ code C++: for (int i = 0; i < 3; i++) { cout << "Data Structures" << endl; } Các tài liệu cũ đã được loại bỏ và thay bằng tài liệu mới.
